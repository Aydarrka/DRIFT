# DRIFT

Instant flash squads nearby — a hackathon MVP for killing boredom in one tap.

## Stack

- Next.js (App Router) + React + TypeScript
- Tailwind CSS v4
- Framer Motion
- Lucide React

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) on a mobile viewport for the best experience.

## User flow

1. **Home (`/`)** — Tap "I'm bored", pick a vibe (Active, Chill, Deep Talk)
2. **Searching (`/searching`)** — Radar animation while mock matching runs (~5s)
3. **Match (`/match`)** — Bottom sheet with squad avatars, Bishkek activity plan, and confirm button
